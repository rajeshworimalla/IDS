===========================================
KALI LINUX - QUICK START GUIDE
Copy this file to Kali Linux
===========================================

STEP 1: Copy the script to Kali Linux
--------------------------------------
Method 1: Copy file "kali-attacks.sh" to Kali
Method 2: Create new file and paste the script content

STEP 2: Make it executable
--------------------------------------
chmod +x kali-attacks.sh

STEP 3: Edit the IP address (if needed)
--------------------------------------
nano kali-attacks.sh
# Change this line: IDS_IP="192.168.100.4"
# Press Ctrl+X, then Y, then Enter to save

STEP 4: Run the script
--------------------------------------
./kali-attacks.sh

STEP 5: Select attack type from menu
--------------------------------------
1 = Port Scan
2 = DoS Attack
3 = Brute Force
4 = R2L Attack
5 = U2R Attack
6 = Mixed Attack
7 = Test ALL (runs all 6 attacks)
8 = Exit

    -d "username=admin&password=wrong$i"

  ssh -o ConnectTimeout=1 fakeuser$i@192.168.100.4 "exit" 2>&1
